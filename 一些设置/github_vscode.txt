1.安装好git软件

2.重启vscode，在代码管理器处点 +号，选择代码文件夹

3.Github上新建的目录，里面不要有readme之类的文件。

4.本地文件夹如果已有文件，则输入命令
git remote add origin https://github.com/wear99/Python_Study.git
git push -u origin master


网站上的文件和本地的一定要有所关联，否则会出错。解决办法：
1.清空网站上目录内的
2.剪切完本地目录的，连接好后，再粘帖回来。

拉取：从github下载到本地；
推送：从本地上传到github

上传过程：
本地的文件先要暂存更改；
然后在上传；



…在本地建立一个新的目录输入以下命令：
echo "# Python_Study" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/wear99/Python_Study.git
git push -u origin master



…在本地已有文件，需要push到网上，则输入以下命令：
git remote add origin https://github.com/wear99/Python_Study.git
git push -u origin master
